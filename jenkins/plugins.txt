# Install all plugins via: Jenkins UI → Manage Jenkins → Plugin Manager

#   java -jar jenkins-cli.jar -s http://localhost:8080 install-plugin <plugin-id>

# Plugin explanations:
#
# PIPELINE PLUGINS
# ─────────────────
# workflow-aggregator        Core Declarative Pipeline support
# pipeline-stage-view        Visual pipeline stages in Jenkins UI
# blueocean                  Modern pipeline visualization UI
# pipeline-utility-steps     readJSON, readFile, etc. utility steps
#
# SOURCE CONTROL
# ──────────────
# git                        Git SCM integration
# github                     GitHub-specific features (PR status, etc.)
# github-branch-source       Multibranch Pipeline support for GitHub
#
# DOCKER
# ──────
# docker-workflow            docker.build(), docker.withRegistry() steps
# docker-plugin              Docker agents (build on Docker containers)
#
# AWS
# ───
# aws-credentials            Store AWS access key + secret securely
# pipeline-aws               withAWS() step, aws CLI integration
#
# SONARQUBE
# ─────────
# sonar                      withSonarQubeEnv() step, Quality Gate
#
# CREDENTIALS
# ───────────
# credentials                Core credentials store
# credentials-binding        withCredentials() step
# ssh-agent                  sshagent() step for SSH key injection
#
# REPORTING
# ─────────
# htmlpublisher              publishHTML() step for coverage reports
# junit                      JUnit test report parsing
# ansicolor                  ANSI color codes in console output
#
# NOTIFICATIONS (optional)

# email-ext                  Extended email notifications
